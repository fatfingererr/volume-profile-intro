## 大綱

1. 機構交易人特性：成本思維
2. 競價拍賣：機構與短線如何達成共識？
3. 機構成本價位，等於高頻率買賣的價位
4. 掌握機構成本價位：時間價格機會(TPO)
5. 市場當前情況(上) 穩定狀態
6. 市場當前情況(下) 失衡狀態
7. 輪廓尾巴：機構投資人的意圖
8. 失敗的競價：單一輸出
9. 外部因素：消息與基本面的影響
10. 市場輪廓的關鍵價位：控制點 (PoC)
11. 歷史的控制點：裸價
12. 接受與拒絕：對控制點的測試
13. 被削弱的控制點：弱高價 (上)
14. 被削弱的控制點：弱高價 (下)
